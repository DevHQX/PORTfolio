# Devraj Portfolio Template

## Overview
A stylish, modern portfolio template built with Framer. This is a static HTML website featuring bold design, smooth interactions, and full customizability. Perfect for creatives to showcase their work with style and impact.

## Project Structure
- `index.html` - Main portfolio page (static HTML from Framer export)
- `server.py` - Python HTTP server with cache-control headers
- `.replit` - Replit configuration

## Technology Stack
- Static HTML/CSS/JavaScript
- Framer-generated content
- Python 3.11 (for serving static files)

## Setup
The site is served using a Python HTTP server on port 5000 with cache-control headers to ensure updates are visible immediately.

## Running the Project
The server starts automatically via the configured workflow and serves the portfolio at `http://0.0.0.0:5000`
